# WellnessHub - Health & Well-Being Platform

A comprehensive health platform connecting communities, health workers, and providers.

## Features
- Health Camp Locator
- Health Reports
- Telemedicine
- Health Education
- Community Feedback
- User Authentication

## Setup

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Tech Stack
- Frontend: React, CSS3
- Backend: Node.js, Express, MongoDB
- Authentication: JWT, bcrypt
- APIs: OpenCage Geocoding

## Project Structure
- `/backend` - Node.js server with Express
- `/frontend` - React application
- Modern UI with responsive design
- Real-time location services
- Secure authentication system
